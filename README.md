# Food Delivery Time Prediction using Machine Learning

## Project Overview
This project predicts food delivery time using machine learning techniques. 
The goal is to analyze different factors such as weather conditions, traffic density, delivery partner ratings, and distance between restaurant and delivery location to estimate the delivery time.

## Problem Statement
Food delivery platforms like Swiggy and Zomato need accurate delivery time predictions to improve customer experience and optimize delivery operations.

This project builds a machine learning model to predict the expected delivery time based on multiple features.

## Dataset
The dataset contains information about:

- Delivery partner details
- Weather conditions
- Road traffic density
- Distance between restaurant and delivery location
- Type of order and vehicle
- Festival and city information

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Handling Missing Values
5. Encoding Categorical Variables
6. Model Training
7. Model Evaluation
8. Feature Importance Analysis

## Machine Learning Models Used
- LinearRegression
- Random Forest Regressor

## Key Insights
- Weather conditions significantly impact delivery time.
- Delivery partner ratings influence delivery efficiency.
- Distance between restaurant and customer affects delivery duration.
- Multiple deliveries assigned to a partner increase delivery time.
